USB 组件文档

USB 组件 使您能够创建 USB 设备 和 USB 主机 应用程序，并且是 的一部分。USB 协议由 USB 组件处理，因此开发人员可以专注于他们的应用需求。

USB 组件的结构

USB 组件 结构如下：

  • USB 主机（仅适用于 MDK 专业版）用于通过 USB 总线与其他 USB 设备外设进行通信。
  • USB 设备 实现可以连接到 USB 主机的设备外设。
  • 支持以下 USB 类：
    • 人机接口设备（HID）
    • 大容量存储类（MSC）
    • 通信设备类（CDC）
    • 音频设备类（ADC） （仅限 USB 设备）
    • 自定义类（用于实现新的或不支持的 USB 类）
    • 可以实现支持多种设备类的 复合 USB 设备。
  • USB 主机和 USB 设备的 为微控制器外设提供接口。
