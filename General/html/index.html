MDK 中间件文档

MDK 是 ARM Cortex-M 系列微控制器的开发环境，包含用于在 C 或 C++ 程序中创建和测试应用程序的所有内容。MDK 分为 MDK 核心和软件包，使得独立于工具的新设备支持和中间件更新成为可能。

MDK 软件包

包含 IDE，ARM 编译器，包安装程序和调试器。软件包安装设备支持和 CMSIS 组件。

MDK 中间件 软件包包含用于 IPv4 和 IPv6 网络的组件，USB 主机和设备通信，用于数据存储的文件系统以及图形用户界面。实时操作系统 CMSIS-RTOS RTX 被所有中间件组件用于任务调度。
