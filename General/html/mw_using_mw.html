中间件文档

本用户指南包含以下章节：

  • 通用步骤 解释了从零开始创建包含中间件组件的项目的必要步骤。
  • 栈，堆和 RTOS 配置 介绍如何配置项目以满足中间件的资源需求。

每个中间件组件都有一个关于以下主题的部分来解释如何创建一个应用程序：

  • 创建应用程序 包含在嵌入式应用程序中使用组件的分步说明。
  • 示例项目 概述软件组件的关键产品功能。它们经过多个评估板的测试，实施和验证，可以作为应用开发的起点。
  • 资源需求 说明每个组件的堆栈，线程和内存资源要求。
  • 参考 记录组件和每个 API 函数的文档。
